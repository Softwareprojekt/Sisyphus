Dieses Verzeichnis enthält vorkonfigurierte Microsoft Visual Studio-Workspaces:
===============================================================================

Microsoft Visual Studio 2010


Root(.sln)
----------

Leerer Workspace (alias "Projektmappe", "Solution"). Ist bereits an das Versionsverwaltungssystem angebunden.

Root.sln enthält folgende wichtige Informationen:

- welche Visual Studio Projekte gibt es?
- wie sind die Abhängigkeiten zwischen diesen Projekten? (wichtig für die korrekte Reihenfolge beim Build!)

Root.sln muss also geändert und wieder eingecheckt werden, wenn:

- neue Visual Studio Projekte hinzukommen
- bestehende Visual Studio Projekte gelöscht oder umbenannt werden
- die Abhängigkeiten zwischen Visual Studio-Projekten geändert werden
